# 🧾 SmartMart POS – Next Gen Billing System

A modern, responsive **Point of Sale (POS) web application** designed for supermarkets and retail stores.

Built with a focus on **clean UI, performance, and real-world usability**.

---

## 🚀 Features

- 📊 Dashboard with real-time analytics
- 💰 Billing system with live total calculation
- 📦 Inventory management (CRUD)
- 📜 Transaction history tracking
- 🎨 Modern glassmorphism UI
- 🌙 Dark / Light mode support
- ⚡ Fast and responsive (mobile + desktop)

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Storage: JSON-based (local storage / files)

---
